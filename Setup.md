# Setup

1. Install pnpm: `npm install -g pnpm`
2. Clone repo: `git clone <git@github.com:flurry101/vibes.git>; cd <your-fork-name>`
3. Run: `pnpm install`
4. Open in VS Code: `code .`
5. Press F5 to run extension
6. Start coding!

## Commands
- `pnpm run watch` - watch mode
- `pnpm run compile` - build once
- `pnpm add <pkg>` - add dependency
